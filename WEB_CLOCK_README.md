# 🌐 Word Clock Web App - Часы словами

**Рабочая альтернатива iOS приложению без Apple Developer Account!**

## 🚀 Быстрый запуск

```bash
# Запустить веб-сервер
./run-web-server.sh
```

Затем откройте: http://localhost:8000/web-clock.html

## 📱 Установка на iPhone (PWA)

1. **Запустите веб-сервер** на вашем компьютере
2. **Найдите IP адрес** вашего компьютера:
   ```bash
   ip addr show | grep "inet " | grep -v 127.0.0.1
   ```
3. **На iPhone откройте Safari** и перейдите:
   ```
   http://ВАШ_IP:8000/web-clock.html
   ```
4. **Добавьте на главный экран**:
   - Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
   - Выберите "Добавить на главный экран"
   - Нажмите "Добавить"

## 🛠️ Как использовать

1. **Запустите сервер** на компьютере в той же WiFi сети, что и iPhone
2. **Откройте на iPhone** по IP адресу
3. **Добавьте на главный экран** для удобства
4. **Наслаждайтесь** часами словами!

## 📋 Системные требования

- ✅ iPhone с Safari (любая модель)
- ✅ iOS 11.3+ (для PWA)
- ✅ WiFi сеть (для первого запуска)
- ✅ Компьютер для запуска сервера

## 🔧 Технические детали

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **PWA**: Service Worker + Web App Manifest
- **Responsive**: Адаптивный дизайн
- **Offline**: Кеширование через Service Worker
