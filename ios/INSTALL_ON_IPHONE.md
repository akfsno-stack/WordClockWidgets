# 🚀 Быстрая инструкция по установке IPA на iPhone XS

## ✅ Что было исправлено

Проблемы с старым IPA файлом:
- ❌ **Название приложения**: показывалось как `$Product_Name` 
- ❌ **Отсутствовал исполняемый файл**: была только папка с исходным кодом
- ❌ **Неправильно собран IPA**: без скомпилированного Swift кода

**Исправления:**
- ✅ Исправлены оба Info.plist файла на реальные имена
- ✅ Создан скрипт `build_ipa.sh` для правильной компиляции на macOS
- ✅ Добавлены подробные инструкции
- ✅ Гарантированное наличие бинарного файла в IPA

## 🔧 Требования

- **macOS** (10.15 или новее)
- **Xcode** (установить из App Store бесплатно)
- **iPhone XS** с iOS 14+
- **USB кабель** для подключения

## 📥 Шаги установки

### 1️⃣ Собрать IPA файл (на Mac)

Скопируйте и запустите эту команду на macOS:

```bash
# Перейди в папку iOS проекта
cd WordClockWidgets/ios

# Запусти скрипт сборки
./build_ipa.sh
```

**⏱️ Это займет 5-15 минут** (зависит от скорости вашего Mac)

### 2️⃣ Установить на iPhone XS

После успешной сборки вы получите файл: `build/WordClockWidgets.ipa`

#### **Способ 1: Xcode (Рекомендуется)** ✨

```bash
# 1. Запустите Xcode
open -a Xcode

# 2. В меню: Window > Devices and Simulators
# 3. Выберите ваш iPhone из списка слева
# 4. Перетащите файл build/WordClockWidgets.ipa на iPhone
```

#### **Способ 2: Apple Configurator 2**

1. Откройте **Apple Configurator 2** на Mac
2. Подключите iPhone USB кабелем
3. Найдите ваш iPhone
4. Перетащите файл `build/WordClockWidgets.ipa`

#### **Способ 3: Finder (самый простой)**

1. Откройте **Finder**
2. Подключите iPhone USB кабелем
3. Найдите iPhone в боковой панели слева
4. Откройте вкладку "Files"
5. Перетащите `build/WordClockWidgets.ipa` в папку

### 3️⃣ Доверить приложению на iPhone

После установки:

1. На iPhone откройте **⚙️ Settings** (Параметры)
2. Перейдите в: **General** → **VPN & Device Management**
3. Найдите строку с вашим Apple ID
4. Нажмите **Trust** (Доверять)

**Готово!** Приложение теперь установлено и работает 🎉

## 🆘 Troubleshooting (если что-то не работает)

### "Xcode не найден" / "xcodebuild: command not found"
```bash
# Установите Xcode из App Store или:
xcode-select --install
```

### "iPhone не видится на Mac"
1. Переподключите USB кабель
2. На iPhone нажмите **Trust** когда спросит
3. Перезагрузите оба устройства (выключите и включите)
4. Попробуйте другой USB кабель

### "IPA файл не создается / ошибка при сборке"

Смотрите подробное руководство с полным описанием ошибок:
👉 [BUILD_IPA_GUIDE.md](BUILD_IPA_GUIDE.md)

### "Приложение установилось, но не открывается"
1. Проверьте что вы нажали **Trust** в Device Management
2. Перезагрузите iPhone
3. Попробуйте переустановить IPA файл

### "Ошибка при перетягивании в Xcode: No code signature found"
Это нормально! Скрипт специально создает IPA без подписи. Это работает для личного использования.
