# 🎯 Word Clock - Решение проблемы установки

## ❌ Проблема с IPA

**iOS приложения требуют code signing для установки на реальные устройства.** Без Apple Developer Program ($99/год) установить нативное приложение на iPhone невозможно - это политика Apple.

## ✅ Решение: Web версия (PWA)

Создал полнофункциональную **Progressive Web App (PWA)** версию часов словами!

### 🚀 Как запустить

```bash
# Запустить веб-сервер
./run-web-server.sh
```

### 📱 Как установить на iPhone

1. **Запустите сервер** на компьютере
2. **Найдите IP адрес** компьютера:
   ```bash
   ip addr show | grep "inet " | grep -v 127.0.0.1
   ```
3. **На iPhone откройте Safari**:
   ```
   http://ВАШ_IP_АДРЕС:8000/web-clock.html
   ```
4. **Добавьте на главный экран**:
   - Кнопка "Поделиться" → "Добавить на главный экран"
   - Нажмите "Добавить"

### ✨ Что работает

- ✅ **Время словами** на русском языке
- ✅ **PWA режим** - выглядит как нативное приложение
- ✅ **Оффлайн** - работает без интернета
- ✅ **Главный экран** - иконка на домашнем экране
- ✅ **Автообновление** каждую секунду
- ✅ **Современный дизайн**

### 🎯 Преимущества

- 🚫 **Нет ограничений Apple** - работает без developer account
- 🚫 **Не нужно code signing**
- 🚫 **Не нужно iTunes/iMazing**
- ✅ **Работает на любом iPhone**
- ✅ **Легко обновлять**
- ✅ **Не занимает место в App Store**

### 📁 Файлы

- `web-clock.html` - Основное приложение
- `sw.js` - Service Worker для PWA
- `manifest.json` - PWA манифест
- `run-web-server.sh` - Скрипт запуска сервера
- `WEB_CLOCK_README.md` - Подробная инструкция

### 🔧 Запуск сервера

Сервер уже запущен в фоне. Если нужно перезапустить:

```bash
# Остановить текущий
pkill -f "python3 -m http.server"

# Запустить заново
./run-web-server.sh
```

### 📱 Тестирование

Откройте в браузере: http://localhost:8000/web-clock.html

На мобильном устройстве используйте IP адрес компьютера вместо localhost.
