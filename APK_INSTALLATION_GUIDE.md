## 📱 Быстрая установка

### Шаг 1: Скачать APK
- Откройте файл `download.html` в браузере
- Нажмите кнопку "Скачать APK"
- Или скачайте файл `WordClockWidgets.apk` напрямую

### Шаг 2: Разрешить установку
На вашем Android устройстве:
1. Откройте **Настройки**
2. Выберите **Приложения** или **Безопасность**
3. Включите **"Неизвестные источники"** или **"Установка из других источников"**
4. Подтвердите разрешение

### Шаг 3: Установить
1. Откройте скачанный файл `WordClockWidgets.apk`
2. Нажмите **"Установить"**
3. Дождитесь завершения установки (10-20 секунд)

### Шаг 4: Запустить
1. Найдите приложение **WordClockWidgets** в меню приложений
2. Откройте его
3. Настройте виджет (цвета, позиция)
4. Добавьте виджет на главный экран

## 🔧 Устранение проблем

### "Установка заблокирована"
**Решение:** Включите "Неизвестные источники" в настройках безопасности

### "Приложение не установлено"
**Решение:**
1. Проверьте свободное место (минимум 50 MB)
2. Перезагрузите устройство
3. Попробуйте скачать APK еще раз

### "Виджет не добавляется"
**Решение:**
1. Долгое нажатие на главный экран
2. Выберите "Виджеты"
3. Найдите "WordClockWidgets"
4. Перетащите на экран

### Приложение вылетает
**Решение:**
1. Перезапустите приложение
2. Проверьте версию Android (минимум 5.0)
3. Очистите кэш приложения
